# Demo Guide (Judges)

## Goal
Show value in 2 minutes.
No setup drama.
No integrations.

## Demo script (2 minutes)
1) **Problem**
   - “Most teams track too much. They don’t move one metric.”

2) **Flash Scan**
   - Fill 5 fields.
   - Submit.
   - Show output: Accelerator + quick wins.

3) **Dashboard**
   - Gear card.
   - 5 engine cards.
   - Accelerator (weekly).
   - Action Bay (3–5 wins).

4) **Alignment**
   - North Star.
   - 3 departments.
   - Each has an aligned goal.

5) **Meetings**
   - Open Pit Stop.
   - Show it produces 3 actions, owners, due dates.

6) **CSV**
   - Export Actions CSV.
   - (Optional) Import back a small change.

Close:
- “DriverOS turns noise into signal. One weekly Accelerator. Clear next steps.”

## Click path (primary)
1. Home → Start Flash Scan
2. Submit Flash Scan
3. Dashboard loads
4. Open Meetings → Pit Stop
5. Export CSV

## Backup path (if live input fails)
1. Load Seed Company A
2. Dashboard opens instantly
3. Run steps 3–5 above

## Common judge questions
### “How is this different from dashboards?”
- We force one Accelerator.
- We give actions, owners, and ETAs.
- We connect every goal to the North Star.

### “Why no integrations?”
- Hackathon scope.
- CSV is universal.
- Integrations are planned later.

### “How do teams use it?”
- Flash Scan for instant wins.
- Full Audit to align departments under North Star.
