# DriverOS Kiro Pack

Drop this `.kiro/` folder into your repo root.

## What you get
- Steering docs: product, scope, UI, domain model, scoring, onboarding, goals, meetings, CSV, tests, security
- Prompts: quickstart, prime, plan-feature, execute, code-review, hackathon review, devlog/readme helpers

## Suggested loop
1. `@quickstart`
2. `@prime`
3. `@plan-feature <slice>`
4. `@execute <plan-file>`
5. `@code-review`
6. `@devlog-update`
7. `@readme-sync`

## Custom agents (optional)
- `.kiro/agents/apex-intake.json`
- `.kiro/agents/driveros-builder.json`
